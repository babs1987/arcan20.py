Игра Арканоид 1.0.
1. Для игры требуется установленный python 3, и модуль pygame 
2. Игра полностью бесплатна
3. Правила:
     Классический Арканоид, управляя клавишами влево, вправо перемещаем платформу: необходимо 
  сбить все блоки в верхней части экрана шаром, если шар попадает не на платформу игра заканчивается
